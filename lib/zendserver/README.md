AutoPhing Zend Server
========================

Zend Server WebAPI targets

## Targets ##

### cacheClear ###
Clear the various Zend Server caches
* Zend Data Cache
* Zend Page Cache
* Zend Optimizer+

## Configuration ##
Minimally set to the following config options:
* zendserver.config.user
* zendserver.config.apiKey

## Dependencies ##
None
